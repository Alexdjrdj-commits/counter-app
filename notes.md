# Counter App Notes

## HTML Notes

- The counter number uses `id="count"` so JavaScript can find it.
- Buttons use `onclick` to run JavaScript functions.

## CSS Notes

- `text-align: center;` centers text.
- `font-size` changes how big text appears.
- `margin` creates space around elements.
- `padding` creates space inside elements.

## JavaScript Notes

- `let count = 0;` creates a variable named `count`.
- `count = count + 1;` increases the number.
- `count = count - 1;` decreases the number.
- `document.getElementById("count")` finds the counter paragraph.
- `.textContent = count` updates what the user sees.
- `if (count > 0)` prevents the counter from going below zero.
